# Phase 1 scope

Phase 1 is the fact-building stage of the onboarding agent.

## Deliverables

1. A reusable repo analyzer script.
2. A structured JSON fact file.
3. Human-readable Markdown onboarding facts.
4. A stable folder layout for later skill and runtime integration.

## Non-goals

Phase 1 does not include:

1. autonomous code editing,
2. multi-agent orchestration,
3. standalone web UI,
4. production-grade RAG infrastructure.

## Success criteria

Phase 1 is successful if a newcomer can quickly answer:

1. what framework this project uses,
2. how the app starts,
3. how routing is initialized,
4. where framework code lives,
5. where business code lives,
6. which business domains exist.
